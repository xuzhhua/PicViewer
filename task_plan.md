# PicViewer - 本地图片浏览器 任务计划

## 项目概述
Web 端本地图片/视频浏览器，支持局域网文件夹和映射驱动器。

## 技术栈
- 后端: Node.js + Express + sharp (缩略图) + ffmpeg (视频转码/缩略图)
- 前端: Vite + React
- 样式: CSS
